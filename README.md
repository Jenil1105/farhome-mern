FarHome is a full-stack web application built using the MERN stack.  
The project focuses on building a scalable backend with structured data models and a responsive frontend for managing listings and related workflows.

This project is currently **under active development**.

## Tech Stack
- **Frontend:** HTML, CSS, EJS
- **Backend:** Node.js, Express.js
- **Database:** MongoDB, Mongoose
- **Other Tools:** Git, npm

## Features
- Server-side rendered views using EJS
- RESTful APIs built with Express for adding, updating, and deleting listings
- MongoDB schemas designed for structured and reliable data handling
- Complete CRUD functionality for listings
- Static asset management for CSS and images
